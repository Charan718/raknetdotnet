RakNet bindings for C#

How to use. 

1. Add RakNetDotNet.dll to reference of your project. 
2. Copy RakNetDotNet.dll and RakNetWrapper.dll in a directory same as .exe 

How to build. 

1. Check out from http://raknetdotnet.googlecode.com/svn/ 
2. RakNet30Beta, swig, Ruby extract to top directory. 
3. Open RakNetDotNET.sln and build.
